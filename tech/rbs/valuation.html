Valuation Model

The value of the KASH token is assessed based on the future value of the Reserve Vault and its potential for staking interest. To calculate its present value, this assessment is adjusted to reflect the project's changing risk over time and the time value of money.

Gold Price​

Although the project anticipates a long-term rise in gold prices, this optimistic forecast is not incorporated into the formulas. For value conversions, such as the valuation of intrinsic value and the calculation of discount factors, the gold price at the time of valuation is used.

  • Pgold,tP_{gold, t}Pgold,t​: The price of gold at time t (USD/oz)

For example, in the Face Value calculation below, the "Future Vault Value" is calculated by predicting the "future quantity of gold to be delivered" but multiplying it by the "current gold price."

The most fundamental metric for evaluating the KASH token's value is its Face Value (FV).

  • FVFVFV: Represents the future value of one KASH token when all vRWA contracts in the vault are fully realized. FVt=EVVtSupplytotalFV_t = \frac{EVV_t}{Supply_{total}}FVt​=Supplytotal​EVVt​​
    • SupplytotalSupply_{total}Supplytotal​: The total supply of KASH (Total Supply, KASH).
    • EVVEVVEVV, Expected Vault Value: The expected future value of the vault. It is the sum of assets already in the vault and the value expected to be realized in the future (USD). EVVt=CVVt+GoldFt×Pgold,tEVV_t = CVV_t + {Gold_F}_t \times P_{gold, t}EVVt​=CVVt​+GoldF​t​×Pgold,t​
      • CVVCVVCVV, Current Vault Value: The value currently realized in the vault (USD).
      • GoldFGold_FGoldF​: The remaining vRWA in the vault to be exchanged for assets of equivalent value to gold in the future (ozt).

This is the Intrinsic Value (IV), which includes the value that can be obtained through KASH. For Phase 1, it is currently calculated considering only the expected staking rewards.

Discounting Future Values​

Future realized value is assessed at its present value by applying discounts for time and risk factors. KASH uses two discount factors:

  1. Time Discount Factor (DFTDF_TDFT​): A factor to convert future value into present value, calculated using the risk-free interest rate.

    DFT,t=1(1+RRF)nDF_{T,t} = \frac{1}{(1 + R_{RF})^n}DFT,t​=(1+RRF​)n1​
    • RRFR_{RF}RRF​: Risk-Free Annual Interest Rate.
    • n: The remaining period from the valuation time t to the project's end (in years).

  2. Risk Discount Factor (DFRDF_RDFR​): A factor that reflects the risk associated with the uncertainty of achieving the project's goals. The discount rate is high in the early stages when risk is high, and it decreases as the project progresses and the acquisition of assets in the vault becomes more certain.

    DFR,t=min⁡(1−BaseTrustt,(CVVtEVVfinal−CRVt)p)+BaseTrustt0≤BaseTrustt≤DFR,t≤1DF_{R,t} = \min(1 - BaseTrust_t, \left(\frac{CVV_t}{EVV_{final} - CRV_t}\right)^p) + BaseTrust_t \\ 0 \le BaseTrust_t \le DF_{R,t} \le 1DFR,t​=min(1−BaseTrustt​,(EVVfinal​−CRVt​CVVt​​)p)+BaseTrustt​0≤BaseTrustt​≤DFR,t​≤1
    • CVVCVVCVV, Current Vault Value: The vault value realized to date. It gradually increases as vRWA is realized.
    • EVVfinalEVV_{final}EVVfinal​: The expected vault value at the end of the project when all gold has been delivered.
    • BaseTrust\text{BaseTrust}BaseTrust: A minimum trust constant for the project. It may be adjusted upwards at key project milestones (e.g., from 0.2 initially to 0.8 after exploration completion).
    • ppp: A coefficient (set value) that adjusts the sensitivity of DFRDF_RDFR​'s change as CVVCVVCVV converges to EVVfinalEVV_{final}EVVfinal​.
    • CRVCRVCRV, Current Reserve Value: The current converted value of the reserve supply (Insurance Pool, Team Supply) in USD. CRVt=min⁡((CISt+CTSt)×FVfinal(1+RRF)n,EVVfinal)CRV_t = \min( (CIS_t + CTS_t) \times \frac{FV_{final}}{(1 + R_{RF})^n}, EVV_{final} )CRVt​=min((CISt​+CTSt​)×(1+RRF​)nFVfinal​​,EVVfinal​)
      • CISCISCIS, Current Insurance Supply: Current token amount in the Insurance Pool (KASH).
      • CTSCTSCTS, Current Team Supply: Current token amount in the Team Supply (KASH).
      • FVfinalFV_{final}FVfinal​: Expected Face Value at the project's end (USD/KASH).

Discounted Value Metrics​

By applying these discount factors, each value of the KASH token is assessed at the present time as follows:

  1. Discounted Face Value (DFVDFVDFV): The future Face Value (FV) discounted to its present value, considering time and risk factors.

    DFVt=FVt×DFT,t×DFR,tDFV_t = FV_t \times DF_{T,t} \times DF_{R,t}DFVt​=FVt​×DFT,t​×DFR,t​

  2. Discounted Intrinsic Value (DIVDIVDIV): The future Intrinsic Value (IV) discounted to its present value, considering time and risk factors.

    DIVt=IVt×DFT,t×DFR,tDIV_t = IV_t \times DF_{T,t} \times DF_{R,t}DIVt​=IVt​×DFT,t​×DFR,t​
