# Docusaurus MDX 유용한 기능 종합 예제 (틸드 코드블록)

이 문서는 Docusaurus에서 자주 사용되는 유용한 MDX 기능들을 종합적으로 보여주는 예제입니다.
각 기능을 통해 문서를 더욱 풍부하고 가독성 높게 만들 수 있습니다.

## 1. Admonitions (주의사항, 노트 등)

독자의 주의를 끌거나 특정 정보를 강조할 때 유용합니다. 다양한 타입의 Admonition을 사용할 수 있습니다.

:::note
이것은 **노트(note)** 입니다. 추가적인 정보를 제공할 때 사용합니다.
- 마크다운 리스트도 내부에 사용 가능합니다.
- 중요한 포인트를 요약할 수 있습니다.
:::

:::tip[유용한 팁 제목]
이것은 **팁(tip)** 입니다. 사용자에게 도움이 될 만한 팁을 전달할 수 있습니다.
Admonition에 사용자 정의 제목을 붙일 수도 있습니다.
:::

:::info
이것은 **정보(info)** 블록입니다. 일반적인 정보를 전달할 때 사용합니다.
예를 들어, 특정 기능의 배경 설명 등을 담을 수 있습니다.
:::

:::warning
이것은 **경고(warning)** 입니다. 사용자에게 주의를 요하는 내용을 전달할 때 사용합니다.
예를 들어, 특정 작업 수행 시 발생할 수 있는 부작용 등을 알릴 수 있습니다.
:::

:::danger[매우 중요!]
이것은 **위험(danger)** 블록입니다. 매우 중요한 경고나 치명적인 결과를 초래할 수 있는 내용을 알릴 때 사용합니다.
데이터 손실 가능성, 보안 취약점 등을 강조할 수 있습니다.
:::

## 2. 접기/펼치기 기능 (Collapsible Content)

HTML의 `<details>`와 `<summary>` 태그를 사용하여 내용을 접거나 펼칠 수 있습니다. 긴 내용을 숨겨 문서의 가독성을 높일 수 있습니다.

<details>
<summary>여기를 클릭하여 자세한 내용 보기 (기본 접힘)</summary>

이 부분은 숨겨져 있다가 클릭하면 펼쳐지는 내용입니다.
마크다운 문법도 여기에 사용할 수 있습니다.

- 리스트 아이템 1
- 리스트 아이템 2

```javascript
// 코드 블록도 넣을 수 있습니다. (실제로는 ```javascript 사용)
function greet(name) {
  console.log(`Hello, ${name}!`);
}
```

</details>

<br /> {/* 줄바꿈을 위해 <br /> 추가 */}

<details open>
<summary>처음부터 펼쳐진 내용 (클릭하여 접기)</summary>

이 내용은 `open` 속성으로 인해 페이지 로드 시 기본적으로 펼쳐져 있습니다.
클릭하면 다시 접을 수 있습니다.

**주의:** 너무 많은 내용을 기본으로 펼쳐두면 페이지 로딩 속도에 영향을 줄 수 있습니다.

</details>

### Admonition 내부에 접기 기능 활용

Admonition 내부에서도 접기/펼치기 기능을 사용할 수 있어, 더욱 구조화된 정보 전달이 가능합니다.

:::tip[고급 팁]
<details>
<summary>세부 설정 방법 보기</summary>

여기에는 고급 사용자를 위한 세부 설정 방법이나 추가 정보가 들어갈 수 있습니다.
예를 들어, 설정 파일의 특정 파라미터에 대한 설명 등을 담을 수 있습니다.

1.  `docusaurus.config.js` 파일을 엽니다.
2.  `themeConfig` 객체 내부에 `colorMode` 설정을 찾습니다.
3.  필요에 따라 `defaultMode`나 `disableSwitch` 값을 변경합니다.

</details>
:::

## 3. 탭 (Tabs)

여러 콘텐츠 블록을 탭 형태로 보여줄 수 있어, 공간을 효율적으로 사용하고 정보를 그룹화할 수 있습니다.
예를 들어, 여러 프로그래밍 언어에 대한 코드 예제를 보여줄 때 유용합니다.

<Tabs groupId="operating-systems">
  <TabItem value="windows" label="Windows" default>
    Windows 사용자 가이드입니다.
    ```batch
    echo "Hello from Windows!"
    ```
  </TabItem>
  <TabItem value="macos" label="macOS">
    macOS 사용자 가이드입니다.
    ```bash
    echo "Hello from macOS!"
    ```
  </TabItem>
  <TabItem value="linux" label="Linux">
    Linux 사용자 가이드입니다.
    ```bash
    echo "Hello from Linux!"
    ```
  </TabItem>
</Tabs>

`groupId`를 사용하면 다른 페이지에 있는 탭과 선택 상태를 동기화할 수 있습니다.

## 4. 코드 블록 (Code Blocks)

다양한 프로그래밍 언어의 코드를 신택스 하이라이팅과 함께 보여줄 수 있습니다.
줄 번호, 특정 줄 강조, 제목 표시 등의 기능도 제공됩니다.

```jsx title="src/components/MyComponent.js" {4,6-8}
import React from 'react';

function MyComponent({ message }) {
  // 이 줄은 4번째 줄로 강조됩니다.
  return (
    <div>
      {/* 다음 3줄(6-8)이 강조됩니다. */}
      <h1>컴포넌트 제목</h1>
      <p>{message}</p>
      <button onClick={() => alert('Clicked!')}>클릭</button>
    </div>
  );
}

export default MyComponent;
```
위 예시는 JSX 코드이며, 4번째 줄과 6-8번째 줄이 강조 표시됩니다. `title` 속성으로 파일명도 표시할 수 있습니다.

## 6. 수학 공식 (KaTeX)

Docusaurus는 KaTeX를 통해 아름다운 수학 공식을 렌더링할 수 있습니다.
이를 사용하려면 `docusaurus.config.js`에 `remark-math`와 `rehype-katex` 플러그인 설정이 필요합니다.

**인라인 공식:** 문장 안에 $E=mc^2$ 와 같이 공식을 넣을 수 있습니다.
라플라스 변환은 $\mathcal{L}\{f(t)\} = F(s) = \int_0^\infty e^{-st} f(t) dt$ 로 정의됩니다.

**블록 공식:** 여러 줄로 된 공식이나 중요한 단독 공식은 `$$`로 감싸서 표현합니다.

$$
\int_a^b f(x)dx = F(b) - F(a)
$$

$$
\frac{1}{\left(\sqrt{\phi \sqrt{5}}-\phi\right) e^{\frac{2}{5}\pi}} = 1+\frac{e^{-2\pi}}{1+\frac{e^{-4\pi}}{1+\frac{e^{-6\pi}}{1+\cdots}}}
$$

## 7. 각주 (Footnotes)

마크다운의 각주 기능을 사용하여 문서 내에서 추가적인 설명이나 출처를 제공할 수 있습니다. 각주는 문서 하단에 자동으로 정리되어 표시됩니다.

예시:
```
이 문장 끝에 각주를 추가합니다.[^1]

[^1]: 여기에 각주 내용을 작성합니다. 예: 이 기능은 Docusaurus에서 기본적으로 지원됩니다.
```

실제 적용:
Docusaurus는 훌륭한 문서화 도구입니다.[^1]

[^1]: 여기에 각주 내용을 작성합니다. 예: 이 기능은 Docusaurus에서 기본적으로 지원됩니다.

## 8. 이미지에 캡션 달기

HTML의 `<figure>`와 `<figcaption>` 태그를 사용하여 이미지에 캡션을 추가할 수 있습니다. 이를 통해 이미지에 대한 설명을 제공할 수 있습니다.

예시:
```html
<figure>
  <img src="https://docusaurus.io/img/docusaurus.png" alt="Docusaurus Logo" />
  <figcaption>그림 1: Docusaurus의 로고입니다.</figcaption>
</figure>
```

실제 적용:
<figure>
  <img src="https://docusaurus.io/img/docusaurus.png" alt="Docusaurus Logo" />
  <figcaption>그림 1: Docusaurus의 로고입니다.</figcaption>
</figure>

## 9. Mermaid 다이어그램

Mermaid를 사용하여 다양한 다이어그램을 그릴 수 있습니다. 예를 들어, 흐름도, 시퀀스 다이어그램, 클래스 다이어그램 등을 쉽게 작성할 수 있습니다.

예시:
```markdown
 Mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

실제 적용:
```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```

이 코드는 간단한 흐름도를 생성합니다. A에서 B와 C로 분기되고, B와 C가 D로 합쳐지는 구조를 나타냅니다.

## 10. 커스텀 MDX 컴포넌트 (고급)

Docusaurus는 MDX를 지원하므로, 직접 만든 React 컴포넌트를 문서 내에서 사용할 수 있습니다.
이를 통해 더욱 동적이고 복잡한 UI를 구현할 수 있습니다. (예: 위에서 보여드린 `<Tabs>` 와 `<TabItem>` 컴포넌트도 `@theme`에서 가져온 React 컴포넌트입니다.)

더 자세한 내용은 [Docusaurus MDX 문서](https://docusaurus.io/docs/markdown-features/react)를 참고하세요.

이 예제 파일이 Docusaurus를 활용한 문서 작성에 도움이 되길 바랍니다!
